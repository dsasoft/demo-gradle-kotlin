# demo-gradle-kotlin

## Build

### Unix
```bash
 ./gradlew clean build
 ```
 
 ### Windows
 ```bash
 gradlew.bat clean build
 ```
 
 ----
 
 ## Docker
 
 ### Build
 ```docker build . -t dsasoft/demo-gradle-kotlin:0.0.1-SNAPSHOT```
 
 ### Run
 ```docker run -it dsasoft/demo-gradle-kotlin:0.0.1-SNAPSHOT```
